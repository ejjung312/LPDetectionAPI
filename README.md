## 📝 소개
한국 차량 번호판 텍스트 인식 API입니다.

### 프로젝트 설명
- 클라이언트 프로그램에서 호출하는 API 서버입니다.
- 감지된 차량번호판 이미지를 받으면 PaddleOCR로 텍스트 인식을 합니다.
- 차량번호판 텍스트를 DB에 저장하고 클라이언트에 반환합니다.


### 데이터 훈련 (/colab)
- Roboflow에 있는 한국 차량번호판을 수집해서 훈련했습니다. (2,341개)
- 100 Epoch 훈련 후 ONNX 모델로 변환합니다.
